Overview

Sora 2 Text To Video generates realistic 10-15 second clips from a single prompt. Control duration, orientation, and watermark behavior for production-ready outputs.

Model sora-2-text-to-video
Pricing 30 credits ($0.15) per 10s video

Inputs

  • prompt Required. Text description.
  • duration 10 or 15 seconds.
  • orientation Landscape or portrait.
  • remove_watermark Boolean flag.
  • character_id_list Coming soon.

Outputs

  • url Video URL.
  • duration Seconds.
  • has_watermark Boolean status.
  • credits_used Billing cost.

Usage Notes

  • Write prompts with action, setting, and style.
  • Portrait works best for social media formats.
  • Watermark removal is billed per generation.

Endpoint

Method POST
Path /v1/sora-2/text-to-video
Auth Bearer API key

Send a JSON payload with the prompt and generation options.

Example Request

curl -X POST https://api.example.com/v1/sora-2/text-to-video \
  -H "Authorization: Bearer $SORA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Aerial shot of a neon-lit city at dusk.",
    "duration": 10,
    "orientation": "landscape",
    "remove_watermark": true
  }'
